## Decisions Made

- Turn classification extracted from safety section so downstream sprints (narration, repair strategy) can inject per-class behavior adjacent to the classification header without modifying safety rules.
- PromptSections uses all-optional fields with null = emit nothing, so every existing call site compiles unchanged with `sections: .{}` default.
- Placeholder insertion points use `_ = ctx.sections.field` to satisfy Zig's unused-variable checker without emitting any output — the actual emission logic will land in Plans 02, 04, and 05.
