1. **FrameSink** — a type-erased writer interface for SSE frame output:
   ```zig
   pub const FrameSink = struct {
       ptr: *anyopaque,
       writeFn: *const fn (ptr: *anyopaque, frame: []const u8) void,
       pub fn write(self: FrameSink, frame: []const u8) void {
           self.writeFn(self.ptr, frame);
       }
   };
   ```

2. **RunEventObserver** — wraps an inner Observer, translates ObserverEvents to RunEvents, and writes them to a FrameSink:
   ```zig
   pub const RunEventObserver = struct {
       inner: Observer,
       sink: FrameSink,
       allocator: std.mem.Allocator,

       const vtable = Observer.VTable{ ... };
       pub fn observer(self: *RunEventObserver) Observer { ... }
   };
   ```

   The recordEvent function:
   - Forwards ALL events to inner observer first (same pattern as NarrationObserver)
   - Then translates applicable events to RunEvent and writes SSE frame via sink:
     - `tool_call_start` -> `RunEvent{ .tool_start = .{ .tool = e.tool } }`
     - `tool_call` -> `RunEvent{ .tool_result = .{ .tool = e.tool, .success = e.success, .duration_ms = e.duration_ms, .output_preview = null } }`
     - `turn_stage` -> `RunEvent{ .progress = .{ .phase = e.stage, .state = "update", .label = stageLabel(e.stage), ... } }`
     - `narration_frame` -> `RunEvent{ .progress = .{ .phase = @tagName(e.frame_type), .state = "update", .label = e.message, .tool = e.tool_name, ... } }`
     - `agent_end` -> `RunEvent{ .done = .{ .session_id = null, .message_id = null, .usage_tokens = e.tokens_used, .cost_usd = null } }`
   - Use toSseFrame from run_event_types to serialize, then sink.write, then free the frame buffer
