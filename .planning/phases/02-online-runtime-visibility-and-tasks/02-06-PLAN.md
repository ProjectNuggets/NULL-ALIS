## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Usage data -> /usage command | Token counts and costs are surfaced to user via slash command |
| Usage data -> SSE done event | Session totals will be included in done event payloads |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-14 | I (Information Disclosure) | UsageReport | accept | Usage data is the user's own data; no cross-tenant exposure. Per-model breakdown reveals provider names which are already visible in agent configuration. |
| T-02-15 | D (Denial of Service) | UsageRuntime | mitigate | MAX_TURNS_TRACKED=1024 ring buffer prevents unbounded memory growth in long sessions |
| T-02-16 | T (Tampering) | recordTurn | accept | Only called from agent turn loop (server-side); no external input path to manipulate usage records |
</threat_model>
