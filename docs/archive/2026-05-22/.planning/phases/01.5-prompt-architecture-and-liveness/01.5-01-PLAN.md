<objective>
Refactor the flat imperative prompt builder in `src/agent/prompt.zig` into composable, independently callable section functions with a `PromptSections` struct that downstream sprints (narration, task decomposition, learning, persona) will populate.

Purpose: The current prompt builder mixes identity, safety, turn classification, and runtime sections into one monolithic function with inline `writeAll` blocks. Downstream REQ-019 through REQ-022 need to inject composable sections (persona calibration, narration rules, learned behavioral facts). This refactor creates the extension points.

Output: Refactored `prompt.zig` with composable sections, `PromptSections` struct with optional fields, `TurnClass` enum, zero regression on existing tests.
</objective>
