# nullALIS v0.1 Declaration

Date: 2026-03-18

This repository state is declared as the `v0.1` baseline for public beta readiness.

## Baseline Intent

- Keep runtime/API behavior stable and testable.
- Avoid importing large upstream behavioral changes directly.
- Apply future improvements via targeted cherry-picks or surgical upserts.

## Frozen Surfaces

- Gateway API contract in `docs/openapi-v1.yaml`.
- Current tenant/runtime behavior on this branch.
- Existing lane/session and diagnostics semantics in production profile.

## Deferred Intake Policy

- No broad sync from upstream branches.
- No large memory-behavior rewrites during this freeze window.
- Only scoped, test-backed fixes with explicit rollout validation.

## Notes

- This declaration does not block future hardening work.
- It defines the starting point for controlled incremental changes.
